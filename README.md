# SayEllo

SayEllo is a modern, full-stack web application that brings people together through real-time communication. It's designed to provide a seamless chatting experience, with a focus on privacy, responsiveness, and user engagement.

## Features

- **User Authentication**: Secure sign-in functionality to keep your conversations private.
- **Real-time Communication**: Instantly send and receive messages without any lag, fostering real-time discussions.
- **Responsive Design**: A beautiful, intuitive interface that adapts to any screen size, ensuring accessibility on mobile devices, tablets, and desktops.
- **Scalable Architecture**: Designed with scalability in mind, allowing for future expansions and integrations.

## Technologies

- **Frontend**: Crafted with React, leveraging the Vite build tool for rapid development and deployment. The application is styled with modern CSS techniques, ensuring a responsive and engaging user interface.
- **Backend**: Powered by Express on Node.js, providing a robust and scalable server-side foundation. RESTful API design ensures seamless data exchange and integration capabilities.
- **Deployment**: Frontend deployed on Vercel/Netlify for efficient hosting and global delivery. The backend is hosted on Heroku, offering easy scaling and management.

## Getting Started

To run SayEllo locally:

1. Clone the repository to your local machine.
2. Navigate to the `frontend` directory, install dependencies with `npm install`, and start the development server with `npm run dev`.
3. In a separate terminal, move to the `backend` directory, install dependencies, and launch the backend server with `npm start`.
4. Open your browser and visit the local address provided to explore SayEllo.
